# SimpleRickOnDroid
Android (Kotlin) for rick and morty api for study purpose only!

This project will have:

<p>-MVVM pattern;</p>
-ROOM for offline database;
-KOIN for DI;
-Picasso for images;
-Async calls (Flow/coroutines/LiveData);
-Recycleviews;
-Navigation (and NavArgs);
-Retrofit for API calls;
-Data Binding;
-Firebase auth;
-Firestore.
